1️⃣ Criar tabelas no Supabase

Acesse seu projeto em supabase.com → SQL Editor → New query, cole o código abaixo e clique em Run ▶.

-- Perfis create table public.profiles ( id uuid references auth.users primary key, email text, role text default 'user', created_at timestamptz default now() ); -- Planilhas dos usuários create table public.sheets ( user_id uuid references auth.users primary key, data jsonb default '{}', cols jsonb default '[]', rows jsonb default '[]', updated_at timestamptz default now() ); -- Planilha Base (template admin) create table public.template ( id int primary key default 1, data jsonb default '{}', fixed jsonb default '{}', cols jsonb default '[]', rows jsonb default '[]', updated_at timestamptz default now() ); -- Segurança (RLS) alter table public.profiles enable row level security; alter table public.sheets enable row level security; alter table public.template enable row level security; create policy "auth" on public.profiles for all using(auth.role()='authenticated'); create policy "auth" on public.sheets for all using(auth.role()='authenticated'); create policy "auth" on public.template for all using(auth.role()='authenticated');

2️⃣ Credenciais do Supabase

Vá em Settings → API no seu projeto. Copie a Project URL e a chave anon public (aba Legacy se não aparecer).
